一：MVCStudy
这是一个基于MVC结构开发的小demo，无需引入任何jar包，纯手工开发，方便理解MVC工作模式。
主要功能：用户在页面上输入两个数字，程序进行计算并将结果返回给用户。

二：SSHPlay
这是一个SSH框架的整合练习，尚处在开发结果
数据库：MySql
需求：
1.实现商品信息的增，删，更新操作。
2：以完成功能，在粗陋的界面情况下完成了商品信息的保存和删除。
使用方法：
fork或者直接下载项目到本地，推荐用IDEA打开，修改一下jdbc.properties文件中的相应的属性即可（用户名以及密码）
